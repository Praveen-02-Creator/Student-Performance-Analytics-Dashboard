# üìä Student Performance Analytics Dashboard  
### Excel-Based Data Analysis Project  

This project analyzes a dataset of **1,000 students** to explore how demographic, socioeconomic, and academic factors influence overall performance.  
Using **Microsoft Excel**, the project includes PivotTables, PivotCharts, calculated fields, slicers, and a fully interactive dashboard.

---

##  Project Objectives
- Analyze factors influencing student academic performance  
- Compare Math, Reading, and Writing score patterns  
- Evaluate the impact of parental education & test preparation  
- Build an interactive Excel dashboard for dynamic insights  

---

##  Dataset Description
The dataset includes:

- Gender  
- Race/Ethnicity  
- Parental Level of Education  
- Lunch Type  
- Test Preparation Course  
- Math, Reading, and Writing Scores  

Additional calculated fields were created to categorize student performance levels.

##  Key Insights
### **Test Preparation Course**
Students who completed the test preparation course scored significantly higher.

### **Parental Education Level**
Higher parental education correlates with higher average student scores.

### **Gender Score Comparison**
- Females scored higher in Reading & Writing  
- Males scored slightly higher in Math  

### **Race/Ethnicity Analysis**
Group E showed the highest overall performance.

### **Subject Score Comparison**
Reading scores were highest on average, followed by Writing, then Math.
